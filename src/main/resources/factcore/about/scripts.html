Scorpio4 - Scripts

JSR-223 is a Java framework for embedding scripts into applications.

Scorpio4 includes Javascript and Groovy by default. Other languages are available, including:

  • Groovy
  • JavaScript
  • AWK
  • BeanShell
  • ejs
  • FreeMarker
  • Jaskell
  • Java
  • Jelly
  • JEP
  • Jexl
  • jst
  • JudoScript
  • JUEL
  • OGNL
  • Pnuts
  • Python
  • Ruby
  • Scheme
  • Sleep
  • Tcl
  • Velocity
  • XPath
  • XSLT
